# Onboarding Illustrations

## Required Images

Please add your illustration images to this folder with the following names:

```
onboarding/
├── intro.png           # Welcome/Intro screen: "Welcome to Scory"
├── slide-1.png         # Onboarding slide 1: "Read Journals Without Headaches"
├── slide-2.png         # Onboarding slide 2: "AI Storyteller for Research"
└── slide-3.png         # Onboarding slide 3: "Learn While Playing Quizzes"
```

## Image Specifications

- **Format**: PNG or JPG (PNG recommended for transparency)
- **Size**: 1024x1024px (square, 1:1 aspect ratio)
- **File size**: < 500KB per image (optimize for mobile)
- **Background**: Transparent or match app background (#F8FAFC for light mode)

## Recommended Tools for Optimization

- **ImageOptim** (Mac): https://imageoptim.com/
- **TinyPNG**: https://tinypng.com/
- **Squoosh**: https://squoosh.app/

## Design Tips

1. Keep illustrations **simple and clear**
2. Use **Scory brand colors** (Indigo #6366F1, Purple #8B5CF6)
3. Maintain **consistent style** across all 4 images
4. Leave **padding** around illustrations (20-30px from edges)
5. Consider **dark mode** compatibility

## After Adding Images

Once you add your images, the app will automatically load them. No code changes needed!

Just make sure the file names match exactly:
- `intro.png` (Welcome screen)
- `slide-1.png` (Read Journals)
- `slide-2.png` (AI Storyteller)
- `slide-3.png` (Learn & Quiz)
